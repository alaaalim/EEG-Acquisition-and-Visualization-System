 🧠 EEG Acquisition and Visualization System

📌 Project Overview  
This project implements a complete EEG data acquisition and visualization system. EEG signals are acquired using Arduino-based hardware, then transmitted to MATLAB for real-time signal processing and visualization. A **custom GUI**, developed collaboratively by our 5-member team, enables user-friendly monitoring. Proteus is used to simulate and validate the hardware design. The system provides real-time acquisition, visualization, and offline analysis of EEG signals.

🎯 Project Objectives  
- Acquire EEG signals using Arduino-based hardware.  
- Visualize EEG waveforms in real-time using MATLAB.  
-- **Implement a collaborative GUI for signal monitoring** ([View GUI here](https://github.com/ahmedayman220/EEG-State-Classification/tree/main/GitHub_GUI)) 
- Simulate and validate hardware design using Proteus.
